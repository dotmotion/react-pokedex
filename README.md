## Setup

```sh
# clone repo
git clone https://github.com/dotmotion/react-pokedex.git
```

```sh
# install dependencies
npm install
```

```sh
# run the project
npm run start
```

- Used [i18N](https://github.com/i18next/react-i18next) for multi language support (_english / spanish_)
